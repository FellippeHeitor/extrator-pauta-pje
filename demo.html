📋 Funcionalidades Mantidas

  • ✅ Extração de processos criminais
  • ✅ Filtragem por dígitos específicos
  • ✅ Extração de data e hora das audiências
  • ✅ Formato de saída CSV compatível

  • ✅ Identificação do período da pauta

  • PDF.js: Biblioteca da Mozilla para leitura de PDFs
  • HTML5 File API: Para seleção e processamento de arquivos

  • JavaScript ES6+: Código modular e moderno
  • Progressive Enhancement: Funciona mesmo em navegadores mais antigos
